Autoxploit — Website Vulnerability Scanner

Project Overview

Autoxploit is a comprehensive security toolkit designed to automate the process of web application vulnerability scanning. Built with a Python backend, it combines multiple scanning modules into a single, cohesive solution. The tool features a clean, web-based graphical user interface (GUI) built with HTML, CSS, and Node.js, allowing for intuitive operation in controlled testing environments.

Architectural Design

The architecture is modular, allowing for independent development and testing of each vulnerability scanner. The Python core handles the scanning logic, request management, and payload generation, while the Node.js server manages the user interface, delivering a responsive and interactive experience for configuring scans and viewing results.

Key Functionality & Modules

  • SQL Injection Scanner: Actively probes web forms and URL parameters using a variety of payloads to detect potential error-based and boolean-based SQL injection flaws.
  • Cross-Site Scripting (XSS) Scanner: Injects non-malicious script payloads into input fields to identify both reflected and stored XSS vulnerabilities by analyzing the server's response.
  • CSRF & File Upload Analyzer: Includes dedicated modules to check for Cross-Site Request Forgery weaknesses by analyzing form tokens and to test for insecure file handling mechanisms that could allow malicious uploads.
  • Reconnaissance Engine: Features a custom network scanner, inspired by Nmap, for initial host discovery, port scanning, and service banner grabbing to map the application's attack surface.

NetInject — Network Interceptor

Project Overview

Developed as part of my advanced Python studies based on the Z-Security curriculum, NetInject is a powerful Man-in-the-Middle (MITM) tool. It's designed to intercept network traffic and modify data packets on the fly to perform targeted actions, such as forcing file downloads on other devices within the network.

Key Techniques Implemented

  • ARP Spoofing: The core of the tool, tricking devices on a local network into sending their traffic through the attacker's machine.
  • Packet Interception: Utilizes netfilter queues to capture and hold packets for analysis and modification before they reach their destination.
  • Dynamic Packet Modification: Uses the Scapy library to parse and rebuild intercepted HTTP packets, replacing legitimate download links with malicious ones.
  • HTTPS Bypass (Proof of Concept): Includes functionality to demonstrate SSL stripping techniques in controlled lab environments.

Technologies Used

Python Linux (Kali) Scapy Netfilter
